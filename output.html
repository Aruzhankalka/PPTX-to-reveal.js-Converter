The connect four game

The connect four game

The game has a grid-layout with columns and rows.

Columns are labeled with letters, rows with numbers

This should be similar for your games

The action is very simple. State the letter for the column of your next piece

For the human player, you can simply type the letter on the keyboard

Here is a typical callout

Computer opponent

Computer opponent

The algorithm for the bot is rather primitive

It tries to find potential winning situations for the opponent and prevent them.

If no such situation is recognized, it simply uses a random column.

It has no own „winning“ algorithm implemented.

In the example, it will put a piece in B or E to prevent three in a row with both ends clear for the winning stone.
